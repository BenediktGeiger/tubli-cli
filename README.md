# ![Est Flag](src/assets/est_flag_icon_40.png) TUBLI CLI

The TUBLI Command Line Interface is a unified tool to query the sonapi API (sonapi.ee)

# Get Started

```
$ npm install -g tubli
$ tubli --help
USAGE
  $ tubli maja
...
```

# Commands

```
tubli maja
tubli maja -m=plain
tubli maja -m=json
tubli maja -m=table
tubli maja -m=table
tubli maja -m=table
tubli house -l=en
tubli house -l=en -m=json
...
```
